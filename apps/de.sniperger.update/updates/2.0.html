Software Update
Details
This update contains a whole new design and a range of new features, including the following:
  • New design
    • App Screen shows apps as icons instead of a list
    • Choose between two styles: iOS Home Screen or iCloud Drive
  • Apps
    • App Icons are generated through a centralized JSON file, enabling third-party apps
    • Icons can be rearranged, added or deleted
    • Apps can receive notifications (including badges)
    • Almost every app has been completely rewritten
    • Home Button always navigates back to Home Screen
  • Start Screen
    • Start Screen displays a short overview over the functionality in Bug Reporter⁸
  • Setup Assistant
    • Setup Assistant helps first-time users set up Bug Reporter⁸ to their needs
    • Assistant automatically shows up after successfully restoring Bug Reporter⁸
  • Tips
    • Tips finally has content again!
    • Content is generated from a centralized JSON file
    • Preliminary compatibility for iPad and desktop
  • Software Update
    • Software Update now also restores Bug Reporter⁸ back to factory settings
    • Previous updates are now displayed in a list
  • User Control Panel
    • MySQL Database users supported out-of-the-box
  • Settings
    • New settings have been added: Language, Background
  • Localization
    • Bug Reporter⁸ supports English and German localization
    • New localizations can be easily added
  • Core
    • JavaScript Core has been completely rewritten (again)
    • Backward compatibility to Bugs filed in Bug Reporter⁷
    • New API methods (details can be found here)
